## 🎯 New Formulas Added (13 formulas)

### Statistics (3 new):
1. ✅ **Arithmetic Mean** - `x̄ = Σx / n`
   - Variables: x (values), n (count)
   - Description: Average of numbers
   - Example: Mean of 2,4,6,8,10 = (2+4+6+8+10)/5 = 6

2. ✅ **Variance** - `σ² = Σ(x-x̄)² / n`
   - Variables: x (values), x̄ (mean), n (count)
   - Description: Measure of data spread
   - Shows how far numbers are from the mean

3. ✅ **Standard Deviation** - `σ = √(Σ(x-x̄)² / n)`
   - Variables: x (values), x̄ (mean), n (count)
   - Description: Square root of variance
   - Most common measure of spread

### Combinatorics (2 new):
4. ✅ **Permutation** - `nPr = n! / (n-r)!`
   - Variables: n (total items), r (items to arrange)
   - Description: Arrangements where order matters
   - Example: 5P3 = 5!/(5-3)! = 60

5. ✅ **Combination** - `nCr = n! / (r! × (n-r)!)`
   - Variables: n (total items), r (items to select)
   - Description: Selections where order doesn't matter
   - Example: 5C3 = 5!/(3!×2!) = 10

### Financial Math (2 new):
6. ✅ **Simple Interest** - `SI = (P × R × T) / 100`
   - Variables: P (principal), R (rate %), T (time in years)
   - Description: Interest = Principal × Rate × Time / 100
   - Example: P=1000, R=5%, T=2 → SI = 100

### Percentage & Profit/Loss (5 new):
8. ✅ **Percentage** - `% = (part / whole) × 100`
   - Variables: part, whole
   - Description: Convert fraction to percentage
   - Example: 25/100 × 100 = 25%

9. ✅ **Profit Percentage** - `P% = ((SP - CP) / CP) × 100`
   - Variables: SP (Selling Price), CP (Cost Price)
   - Description: Profit percentage calculation
   - Example: CP=100, SP=120 → P% = 20%

10. ✅ **Loss Percentage** - `L% = ((CP - SP) / CP) × 100`
    - Variables: CP (Cost Price), SP (Selling Price)
    - Description: Loss percentage calculation
    - Example: CP=100, SP=80 → L% = 20%

11. ✅ **Discount** - `D = MP - SP`
    - Variables: MP (Marked Price), SP (Selling Price)
    - Description: Discount amount
    - Example: MP=500, SP=400 → D = 100

12. ✅ **Discount Percentage** - `D% = ((MP - SP) / MP) × 100`
    - Variables: MP (Marked Price), SP (Selling Price)
    - Description: Discount percentage on marked price
    - Example: MP=500, SP=400 → D% = 20%

### Speed, Distance, Time (3 new):

    - Variables: Speed, Time
    - Description: Calculate distance
    - Example: Speed=60km/hr, Time=3hr → Distance = 180km

15. ✅ **Time** - `Time = Distance / Speed`
    - Variables: Distance, Speed
    - Description: Calculate time
    - Example: Distance=200km, Speed=50km/hr → Time = 4hr

## 🎓 Practical Examples

### Example 1: Calculate Mean
```
Data: 10, 20, 30, 40, 50
Formula: x̄ = Σx / n
Calculation: (10+20+30+40+50) / 5 = 150/5 = 30
Result: Mean = 30
```

### Example 2: Permutation
```
Question: How many ways to arrange 3 books from 5?
Formula: nPr = n! / (n-r)!
Given: n=5, r=3
Calculation: 5P3 = 5! / (5-3)! = 120/2 = 60
Result: 60 ways
```

### Example 3: Simple Interest
```
Principal: ₹10,000
Rate: 8% per year
Time: 3 years
Formula: SI = (P × R × T) / 100
Calculation: (10000 × 8 × 3) / 100 = 2400
Result: Interest = ₹2,400
```

### Example 4: Profit Percentage
```
Cost Price: ₹500
Selling Price: ₹650
Formula: P% = ((SP - CP) / CP) × 100
Calculation: ((650 - 500) / 500) × 100 = 30%
Result: Profit = 30%
```

### Example 5: Speed Calculation
```
Distance: 300 km
Time: 5 hours
Formula: Speed = Distance / Time
Calculation: 300 / 5 = 60
Result: Speed = 60 km/hr
```

## 💡 Tips for Using These Formulas

### Statistics:
- Use Mean for average
- Use Standard Deviation to measure consistency
- Lower SD = more consistent data

### Combinatorics:
- Use Permutation when order matters (e.g., passwords)
- Use Combination when order doesn't matter (e.g., lottery)

### Financial:
- Simple Interest: Linear growth
- Compound Interest: Exponential growth
- Always better for investments

### Business Math:
- Profit = SP > CP
- Loss = SP < CP
- Discount reduces Marked Price

### Speed-Distance-Time:
- Remember: Speed × Time = Distance
- Units must match (km/hr with km and hr)
