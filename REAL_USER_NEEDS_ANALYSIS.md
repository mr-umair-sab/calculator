
   ```

   Solve for x and y
   ```

   ```
   2x + 5 > 15
   Solve: x > ?
   ```

## 📚 **Class 11-12 (Advanced)**

### ✅ What They Use (You Have):
- Trigonometry (sin, cos, tan rules)
- Calculus basics
- Complex formulas
- Chemistry formulas

### ❌ What's MISSING:

1. **Logarithm Calculator** ⚠️ CRITICAL
   ```
   log₂(8) = ?
   ln(e³) = ?
   Change of base
   Logarithm properties
   ```

2. **Exponential Calculator** ⚠️
   ```
   e^x calculations
   a^b for any a, b
   ```

3. **Sequence & Series** ⚠️
   ```
   AP: Find nth term, sum
   GP: Find nth term, sum
   Sum of n natural numbers
   Sum of squares: 1² + 2² + ... + n²
   ```

4. **Binomial Theorem** ⚠️
   ```
   (a + b)ⁿ expansion
   Find specific term
   Binomial coefficients
   ```

5. **Determinant Calculator** ⚠️ CRITICAL
   ```
   2×2 determinant
   3×3 determinant
   Cramer's rule
   ```

6. **Vector Calculator** ⚠️ CRITICAL
   ```
   Dot product: a⃗·b⃗
   Cross product: a⃗×b⃗
   Magnitude: |a⃗|
   Unit vector
   Angle between vectors
   ```

7. **Limits Calculator** ⚠️
   ```
   lim(x→0) sin(x)/x
   lim(x→∞) (1 + 1/x)^x
   ```

8. **Differentiation** ⚠️ CRITICAL
   ```
   d/dx(x²) = 2x
   d/dx(sin x) = cos x
   Product rule
   Quotient rule
   Chain rule
   ```

9. **Integration** ⚠️ CRITICAL
   ```
   ∫x² dx = x³/3
   ∫sin x dx = -cos x
   Definite integrals
   ```

10. **Probability Calculator** ⚠️
    ```
    P(A∪B) = P(A) + P(B) - P(A∩B)
    Conditional probability
    Bayes theorem
    ```
