# 🧮 Smart Calculator - Complete Multi-Mode Calculator

A comprehensive calculator web application designed for students from Class 5 to BS/University level.

---

## 🌟 Features

### **6 Calculator Modes:**
1. **📚 5th Class** - Basic arithmetic
2. **🔬 10th Class** - Scientific functions
3. **🧪 12th Class** - Advanced scientific
4. **🎓 BS** - Quadratic solver, Matrix, GPA
5. **🔧 Utilities** - 19 specialized calculators
6. **📐 Formulas** - 109 formula library

---

## 🚀 Quick Start

### **Option 1: Direct Use**
1. Open `public/index.html` in any browser
2. Select your calculator mode
3. Start calculating!

### **Option 2: Development**
```bash
# Install dependencies
npm install

# Build CSS
npm run build

# Open in browser
open public/index.html
```
