## 🚀 **What Was Added in Phase 3**

### **1. Basic Calculus Calculator** 📊
**Location:** Utilities → Calculus

**Features:**
- ✅ **Differentiation**
  - Power rule: d/dx(x^n) = nx^(n-1)
  - Trigonometric: sin, cos, tan
  - Exponential: e^x
  - Logarithmic: ln(x)
  - Simple sums

- ✅ **Integration**
  - Power rule: ∫x^n dx = x^(n+1)/(n+1) + C
  - Trigonometric: sin, cos
  - Exponential: e^x
  - Logarithmic: 1/x
  - Simple sums

**Examples:**
```
Differentiation:
d/dx(x²) = 2x
d/dx(x³ + 2x) = 3x² + 2
d/dx(sin x) = cos x
d/dx(e^x) = e^x
d/dx(ln x) = 1/x

Integration:
∫x² dx = x³/3 + C
∫sin x dx = -cos x + C
∫e^x dx = e^x + C
∫1/x dx = ln|x| + C
```

**Use Cases:**
- Class 12: Calculus basics
- BS: Engineering calculations
- Physics problems
- Optimization

### **3. Probability Calculator** 🎲
**Location:** Utilities → Probability

**Features:**
- ✅ **Binomial Probability**
  - P(X = k) for n trials
  - Success probability p
  - Combinations nCk

- ✅ **Union Probability**
  - P(A ∪ B) = P(A) + P(B) - P(A ∩ B)

**Examples:**
```
Binomial:
n=10, k=3, p=0.5
P(X=3) = 0.117188 (11.72%)

Union:
P(A)=0.3, P(B)=0.4, P(A∩B)=0.1
P(A∪B) = 0.6
```

**Use Cases:**
