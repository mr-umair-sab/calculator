## 📊 What Was Added

### 1️⃣ **Fraction Calculator** ✅
**Location:** Utilities → 🔢 Fractions

**Features:**
- Add fractions: 2/3 + 4/5
- Subtract fractions: 5/6 - 1/3
- Multiply fractions: 3/4 × 2/5
- Divide fractions: 1/2 ÷ 3/4
- Simplify fractions: 6/8 = 3/4
- Automatic simplification
- Decimal conversion

**Example:**
```
Input: 2/3 + 4/5
Result: 22/15
Decimal: 1.466667
```

---

### 2️⃣ **LCM & HCF Calculator** ✅
**Location:** Utilities → 🔢 LCM/HCF

**Features:**
- Find LCM (Least Common Multiple)
- Find HCF/GCD (Highest Common Factor)
- Works with 2 or 3 numbers
- Shows verification

**Example:**
```
Input: 12, 18
HCF: 6
LCM: 36
Verification: 12 × 18 = 216 = 6 × 36
```

---

### 3️⃣ **Simultaneous Equations Solver** ✅
**Location:** Utilities → 📐 Equations

**Features:**
- Solve 2×2 linear equations
- Format: a₁x + b₁y = c₁ and a₂x + b₂y = c₂
- Shows solution for x and y
- Verification included
- Detects no solution/infinite solutions

**Example:**
```
Equation 1: 2x + 3y = 12
Equation 2: x - y = 1

Solution:
x = 3
y = 2

Verification:
2(3) + 3(2) = 12 ✓
3 - 2 = 1 ✓
```

---

### 4️⃣ **Logarithm Calculator** ✅
**Location:** Utilities → 📊 Logarithm

**Features:**
- log₁₀ (Common logarithm)
- ln (Natural logarithm)
- log₂ (Binary logarithm)
- logₓ (Custom base)
- Verification included

**Example:**
```
Input: log₂(8)
Result: 3
Verification: 2³ = 8 ✓
```

---

### 5️⃣ **Vector Calculator** ✅
**Location:** Utilities → ➡️ Vectors

**Features:**
- Add vectors: A + B
- Subtract vectors: A - B
- Dot product: A · B
- Cross product: A × B (3D)
- Magnitude: |A|
- Angle between vectors

**Example:**
```
Vector A = (3, 4, 0)
Vector B = (1, 2, 0)

Dot Product: A · B = 11
Magnitude: |A| = 5
Angle: θ = 11.31°
```
