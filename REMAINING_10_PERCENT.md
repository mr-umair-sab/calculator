#### 1. **Differentiation Calculator** ⭐⭐⭐⭐⭐
**Difficulty:** Hard  
**Time:** 1 week  
**Impact:** Very High

**Features Needed:**
```
Basic Differentiation:
- d/dx(xⁿ) = nxⁿ⁻¹
- d/dx(sin x) = cos x
- d/dx(cos x) = -sin x
- d/dx(eˣ) = eˣ
- d/dx(ln x) = 1/x

Rules:
- Power rule
- Product rule: d/dx(uv) = u'v + uv'
- Quotient rule: d/dx(u/v) = (u'v - uv')/v²
- Chain rule: d/dx(f(g(x))) = f'(g(x))·g'(x)

Examples:
- d/dx(x³) = 3x²
- d/dx(x²·sin x) = 2x·sin x + x²·cos x
- d/dx(eˣ·ln x) = eˣ·ln x + eˣ/x
```

**Why Important:**
- Class 12 & BS students need this daily
- Engineering calculations
- Physics problems
- Optimization problems
