# ⚡ Quick Start Guide - Formula Feature

## 🚀 5 Minutes Setup

### Step 1: Open Calculator (30 seconds)
```
1. Open your browser
2. Go to calculator website
3. Click "📐 Formulas" button
```

### Step 2: Explore Library (1 minute)
```
1. You'll see 25+ pre-loaded formulas
2. Browse by category: Physics, Math, Chemistry, etc.
3. Use search bar to find specific formulas
```

### Step 3: Use a Formula (2 minutes)
```
1. Click "🧮 Use Formula" on any formula card
2. Enter values for all variables
3. Click "Calculate"
4. See your result!
```

### Step 4: Save Your Favorites (1 minute)
```
1. Click "💾 Save" on formulas you use often
2. They'll appear in "My Formulas" tab
3. Access them anytime!
```

### Step 5: Add Custom Formula (30 seconds)
```
1. Go to "💾 My Formulas" tab
2. Fill in the form:
   - Name: Your formula name
   - Expression: The formula
   - Variables: Comma separated (a,b,c)
   - Category: Choose one
3. Click "➕ Save Formula"
```

## 📝 Quick Examples

### Example 1: Calculate Circle Area
```
Formula: A = π × r²
Input: r = 5
Result: A = 78.54
```

### Example 2: Calculate Velocity
```
Formula: v = u + a × t
Input: u = 10, a = 2, t = 5
Result: v = 20
```

### Example 3: Calculate Force
```
Formula: F = m × a
Input: m = 50, a = 10
Result: F = 500
```

## 🔥 Most Used Formulas

### Physics:
- F = m × a (Force)
- KE = 0.5 × m × v² (Kinetic Energy)
- V = I × R (Ohm's Law)

### Math:
- A = π × r² (Circle Area)
- c² = a² + b² (Pythagorean)
- Quadratic Formula

### Geometry:
- Rectangle Area: A = l × w
- Triangle Area: A = 0.5 × b × h
- Sphere Volume: V = (4/3) × π × r³
